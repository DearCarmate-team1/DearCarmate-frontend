# 차량 목록 조회

URI: /cars
메소드: GET
분류: 차량

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `page` : number (현재 페이지 번호)
- `pageSize` : number (페이지당 아이템 수)
- `status`: possession | contractProceeding | contractCompleted (차량 상태)
- `searchBy` : carNumber | model (검색 기준)
- `keyword` : string (검색어)
