# 계약서 업로드

URI: /contractDocuments/upload
메소드: POST
분류: 계약서

**Request header**

- Authorization: Bearer `token`
- Content-Type: `multipart/form-data`

**Request body**

```json
{
	"contractDocument": "File"
}
```

**Responses**

200 OK

```json
{
	"contractDocumentId": 1
}
```

401 Unauthorized

```jsx
{
	"message": "로그인이 필요합니다"
}
```
