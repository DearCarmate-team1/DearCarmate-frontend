# 계약서 추가 시 계약 목록 조회

URI: /contractDocuments/draft
메소드: GET
분류: 계약서

**Request header**

- Authorization: Bearer `token`

**Responses**

200 OK

```json
[
	{
		"id": 1,
		"data": "그랜저 - 김고객 고객님"
	},
	{
		"id": 2,
		"data": "K3 - 홍길동 고객님"
	},
	...
]
```

400 Bad Request

```jsx
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```jsx
{
	"message": "로그인이 필요합니다"
}
```
