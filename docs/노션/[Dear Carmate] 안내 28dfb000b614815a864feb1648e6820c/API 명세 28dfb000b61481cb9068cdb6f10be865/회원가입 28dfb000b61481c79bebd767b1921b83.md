# 회원가입

URI: /users
메소드: POST
분류: 유저

**Request body**

```json
{
	"name": "string",
	"email": "string",
	"employeeNumber": "string",
	"phoneNumber": "string",
	"password": "string",
	"passwordConfirmation": "string",
	"company": "string",
	"companyCode": "string"
}
```

**Responses**

201 Created

```json
{
  "id": 1,
  "name": "string",
	"email": "string",
	"employeeNumber": "string",
	"phoneNumber": "string",
	"imageUrl": "string",
	"isAdmin": true,
	"company": {
		"companyCode": "string"
	}
}
```

400 Bad Request

```jsx
{
	"message": "비밀번호와 비밀번호 확인이 일치하지 않습니다"
}
```

409 Conflict

```json
{
	"message": "이미 존재하는 이메일입니다"
}
```
