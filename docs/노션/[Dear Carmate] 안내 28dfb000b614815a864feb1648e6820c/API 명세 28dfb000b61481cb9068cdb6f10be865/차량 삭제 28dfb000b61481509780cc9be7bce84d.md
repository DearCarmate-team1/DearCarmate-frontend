# 차량 삭제

URI: /cars/{carId}
메소드: DELETE
분류: 차량

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `carId` : number (차량 ID)

**Responses**

200 OK

```json
{
  "message": "차량 삭제 성공"
}
```

400 Bad Request

```json
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```json
{
	"message": "로그인이 필요합니다"
}
```

404 Not Found

```json
{
  "message": "존재하지 않는 차량입니다"
}
```
