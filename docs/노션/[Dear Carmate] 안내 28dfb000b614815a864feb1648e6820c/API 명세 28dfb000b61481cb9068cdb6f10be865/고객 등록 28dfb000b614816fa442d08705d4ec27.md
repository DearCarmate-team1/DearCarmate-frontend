# 고객 등록

URI: /customers
메소드: POST
분류: 고객

**Request header**

- Authorization: Bearer `token`

**Request body**

```json
{
	"name": "string",
	"gender": "male | female",
	"phoneNumber": "string",
	"ageGroup": "10대 | 20대 | 30대 | 40대 | 50대 | 60대 | 70대 | 80대",
	"region": "서울 | 경기 | 인천 | 강원 | 충북 | 충남 | 세종 | 대전 | 전북 | 전남 | 광주 | 경북 | 경남 | 대구 | 울산 | 부산 | 제주",
	"email": "string",
	"memo": "string"
}
```

**Responses**

201 Created

```json
{
	"id": 123,
	"name": "string",
	"gender": "male | female",
	"phoneNumber": "string",
	"ageGroup": "10대 | 20대 | 30대 | 40대 | 50대 | 60대 | 70대 | 80대",
	"region": "서울 | 경기 | 인천 | 강원 | 충북 | 충남 | 세종 | 대전 | 전북 | 전남 | 광주 | 경북 | 경남 | 대구 | 울산 | 부산 | 제주",
	"email": "string",
	"memo": "string",
	"contractCount": 0
}
```

400 Bad Request

```jsx
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```json
{
	"message": "로그인이 필요합니다"
}
```
