# 내 정보 조회

URI: /users/me
메소드: GET
분류: 유저

**Request header**

- Authorization: Bearer `token`

**Responses**

200 OK

```json
{
  "id": 1,
  "name": "string",
	"email": "string",
	"employeeNumber": "string",
	"phoneNumber": "string",
	"imageUrl": "string",
	"isAdmin": true,
	"company": {
		"companyCode": "string"
	}
}
```

400 Bad Request

```jsx
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```json
{
	"message": "로그인이 필요합니다"
}
```

404 Not Found

```json
{
  "message": "존재하지 않는 유저입니다"
}
```
