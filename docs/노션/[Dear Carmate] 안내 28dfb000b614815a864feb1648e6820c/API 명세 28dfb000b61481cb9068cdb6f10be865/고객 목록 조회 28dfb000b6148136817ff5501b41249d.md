# 고객 목록 조회

URI: /customers
메소드: GET
분류: 고객

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `page` : number (현재 페이지 번호)
- `pageSize` : number (페이지당 아이템 수)
- `searchBy` : name | email (검색 기준)
- `keyword` : string (검색어)
