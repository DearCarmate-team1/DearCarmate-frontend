# 유저 삭제

URI: /users/{userId}
메소드: DELETE
분류: 유저

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `userId` : number (유저 ID)

**Responses**

200 OK

```json
{
  "message": "유저 삭제 성공"
}
```

400 Bad Request

```json
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```json
{
	"message": "관리자 권한이 필요합니다"
}
```

404 Not Found

```json
{
  "message": "존재하지 않는 유저입니다"
}
```
