# 회사 수정

URI: /companies/{companyId}
메소드: PATCH
분류: 회사

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `companyId` : number (회사 ID)

**Request body**

```json
{
	"companyName": "string",
	"companyCode": "string"
}
```

**Responses**

200 OK

```json
{
	"id": 1,
	"companyName": "string",
	"companyCode": "string",
	"userCount": 3
}
```

400 Bad Request

```jsx
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```jsx
{
	"message": "관리자 권한이 필요합니다"
}
```

404 Not Found

```json
{
  "message": "존재하지 않는 회사입니다"
}
```
