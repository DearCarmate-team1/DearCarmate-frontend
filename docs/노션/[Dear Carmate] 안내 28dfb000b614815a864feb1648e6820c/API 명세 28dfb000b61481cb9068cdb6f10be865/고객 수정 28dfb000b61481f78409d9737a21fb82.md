# 고객 수정

URI: /customers/{customerId}
메소드: PATCH
분류: 고객

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `customerId` : number (고객 ID)

**Request body**

```json
{
	"name": "string",
	"gender": "male | female",
	"phoneNumber": "string",
	"ageGroup": "10대 | 20대 | 30대 | 40대 | 50대 | 60대 | 70대 | 80대",
	"region": "서울 | 경기 | 인천 | 강원 | 충북 | 충남 | 세종 | 대전 | 전북 | 전남 | 광주 | 경북 | 경남 | 대구 | 울산 | 부산 | 제주",
	"email": "string",
	"memo": "string"
}
```

**Responses**

200 OK (성공)

```jsx
{
	"id": 123,
	"name": "string",
	"gender": "male | female",
	"phoneNumber": "string",
	"ageGroup": "10대 | 20대 | 30대 | 40대 | 50대 | 60대 | 70대 | 80대",
	"region": "서울 | 경기 | 인천 | 강원 | 충북 | 충남 | 세종 | 대전 | 전북 | 전남 | 광주 | 경북 | 경남 | 대구 | 울산 | 부산 | 제주",
	"email": "string",
	"memo": "string",
	"contractCount": 0
}
```

400 Bad Request

```json
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```json
{
	"message": "로그인이 필요합니다"
}
```

404 Not Found

```json
{
  "message": "존재하지 않는 고객입니다"
}
```
