# 고객 삭제

URI: /customers/{customerId}
메소드: DELETE
분류: 고객

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `customerId` : number (고객 ID)

**Responses**

200 OK

```json
{
  "message": "고객 삭제 성공"
}
```

400 Bad Request

```json
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```json
{
	"message": "로그인이 필요합니다"
}
```

404 Not Found

```json
{
  "message": "존재하지 않는 고객입니다"
}
```
