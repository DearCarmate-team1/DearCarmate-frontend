# 계약 삭제

URI: /contracts/{contractId}
메소드: DELETE
분류: 계약

**Request header**

- Authorization: Bearer `token`

**Parameters**

- `contractId` : number (계약 ID)

**Responses**

200 OK

```json
{
  "message": "계약 삭제 성공"
}
```

400 Bad Request

```json
{
	"message": "잘못된 요청입니다"
}
```

401 Unauthorized

```json
{
	"message": "로그인이 필요합니다"
}
```

403 Forbidden

```json
{
	"message": "담당자만 삭제가 가능합니다"
}
```

404 Not Found

```json
{
  "message": "존재하지 않는 계약입니다"
}
```
